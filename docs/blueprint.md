# **App Name**: HomeServ Connect

## Core Features:

- User Authentication & Profiles: Secure registration, login, and basic profile management for Customer and Worker roles.
- Service Request Creation: Customers can easily post new service requests with details about the job required (e.g., plumbing, electrical).
- AI Job Description Tool: An AI tool that assists customers in clearly describing their service needs, suggesting common details or clarifying questions.
- Job Browsing & Acceptance (Workers): Workers can view available service requests, filter by type or location, and accept jobs.
- Job Status Tracking & Updates: Workers can update the status of ongoing jobs (e.g., accepted, in progress, completed), and customers can view these updates.
- Admin Analytics & User Management: A dashboard for admins to view high-level analytics like active jobs and total users, and perform actions like banning reported users.

## Style Guidelines:

- Primary color: A reliable, cool teal shade (#2688D9) to evoke trust and efficiency.
- Background color: A very light, desaturated blue-gray (#EBF1F6) providing a clean, airy canvas.
- Accent color: A vibrant, darker teal (#089496) to highlight key actions and notifications.
- Headline and Body Font: 'Inter', a modern grotesque sans-serif, offering excellent legibility and a neutral, professional feel for all content.
- Use a consistent set of simple, line-art icons that are clear and intuitive for service categories, user roles, and job statuses.
- Adopt a clean, card-based layout for service listings and job cards, ensuring responsive design across all devices for ease of use.
- Implement subtle, fast-paced transitions for status updates and navigation elements, providing smooth feedback without distracting the user.
